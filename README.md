# Cloud Computing & Big Data

Documentación completa del módulo 05MIND - Universidad Internacional de Valencia

## 📚 Contenido

Este sitio cubre 8 bloques fundamentales:

- **Fundamentos del Dato**: Economía, gobierno y fuentes oficiales
- **Cloud Computing**: AWS, Azure, GCP, DevOps y DataOps
- **Data Lakes**: Arquitecturas y almacenamiento distribuido
- **Big Data**: Fundamentos y minería de datos (CRISP-DM)
- **OLTP vs OLAP**: Sistemas transaccionales y analíticos
- **Infraestructura**: Hadoop, Spark, Kafka, Airflow
- **Metodologías Avanzadas**: KDD, ASUM-DM, TDSP
- **Gobierno de Datos**: Calidad, RGPD y anonimización
